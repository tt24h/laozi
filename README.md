### 老子道德经：原文、译文、注释.pdf

本文使用的原文来自 [老子道德经：四种原文表.pdf](https://github.com/tt24h/daodejing) 中的各表首行文本。

各表首行文本校订自王弼本、帛书本、郭简本及北大本，优先选用古早版本的文本，例外少。

使用王弼本分章方案，配有译文、注释及笔记。注释以文义理解为主，不包含大量字典释义。仓库内有：字典与字典工具的介绍文档，生成本 PDF 的 Typst 格式源码。

注意：仅在编写期间，笔者对文义的理解就经历了多次转变，不排除以后还会推翻文中的观点。故本文仅供参考。

### 下载 PDF：

在 **Releases** 内下载 `laozi.pdf` 。

### 编辑、编译 pdf 文件：

1. 注册免费账户：https://typst.app/ 
2. 新建项目：在 Dashboard 页面，点击 Empty document 大按钮。
3. 上传资源：将本地「编译用源码/」里的所有内容，上传至新建的项目。
4. 指定预览文件：在侧边栏选中 `main.typ` 文件，右键设为预览文件。
5. 完成：导出 pdf。
