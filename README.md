### 老子道德经：原文译文注释

本文原文部分来自 [老子道德经：四种原文表.pdf](https://github.com/tt24h/daodejing) 中的各表首行文本。

各表首行文本校订自王弼本、帛书本、郭简本及北大本，优先选用古早版本的文本，例外少。

使用王弼本分章方案，配有译文注释和笔记。

注释以文义理解为主，不包含大量字典释义。

仓库内有字典与字典工具的介绍文档。

**注意**：在编写期间，笔者对文义的理解就经历了多次转变，不排除以后还会推翻文中的观点。本文仅供参考。
