### 老子道德经：原文译文注释

本文原文部分来自 [老子道德经：四种原文表.pdf](https://github.com/tt24h/daodejing) 中的各表首行文本。

各表首行文本校订自王弼本、帛书本、郭简本及北大本，优先选用古早版本的文本，例外少。

使用王弼本分章方案，配有译文注释和笔记。

注释以文义理解为主，不包含大量字典释义。

仓库内有：字典与字典工具的介绍文档。

**注意**：在编写期间，笔者对文义的理解就经历了多次转变，不排除以后还会推翻文中的观点。本文仅供参考。

### 阅读、下载：

提供 Markdown 和 PDF **两个版本**，使用同个数据源生成，内容一致。

其中 Markdown 版提供两个文件下载，二者排版逻辑不同：

1. **章内原文集中**：章标题 → 整章原文 → 一段译文及其注释等 → 一段译文及其注释等 → … → 章标题。
2. **章内原文分散**：章标题 → 一段原文及其译文注释等 → 一段原文及其译文注释等 → … → 章标题。

章内原文集中，整章原文置于章标题下；章内原文分散，每段原文紧挨其译文。

* **Markdown 版**：[查看集中排版](./章内原文集中排版.md)｜[查看分散排版](./index.md)

   * 集中排版的文件：`./章内原文集中排版.md`
   * 分散排版的文件：`./index.md` ，或 [浏览基于此的网页](https://tt24h.github.io/laozi/)。
   
* **PDF 版**：[下载PDF](https://github.com/tt24h/laozi/releases/latest/download/laozi.pdf)
   
   * 文件：Releases → `laozi.pdf`
   * 使用 集中排版。
   * 在 工具/ 内有 编辑、编译 pdf 的指南。
   
* **数据源**：[查看文件](./PDF编译用源码/数据.txt)

   * 文件：`./PDF编译用源码/数据.txt`
