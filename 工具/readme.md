文件夹内文件说明：
=====

1. 从「数据.txt」还原原文.html：

	* **作用**：提取「数据.txt」中的原文，用于和《老子道德经：四种原文表》中“校订文”核对。
	* 在上方文本框粘贴「数据.txt」的所有内容。
	* 下方文本框会实时显示还原的原文。
	* 使用方法：将 html 文件下载至本地，用浏览器打开。
	
2. 从「数据.txt」生成Markdown版.html：

	* **作用**：解析「数据.txt」数据，生成 Markdown 版文本。
	* 在上方文本框粘贴「数据.txt」的所有内容。
	* 下方文本框会实时显示相应的 Markdown 版文本。
	* 可以选择两种输出类型：章内原文集中型、章内原文分散性。
	* 使用方法：将 html 文件下载至本地，用浏览器打开。
	
3. 编辑、编译PDF简短指南.md

    * **作用**：简短介绍  Typst 编译 PDF 的方法。
    * 介绍了 本地或在线编译方式、更改字体、输出 章内原文集中型、章内原文分散性 文档。
