从 Typst 代码到 pdf 至少有两种途径：1. 本地编译，2. 在线编译。

### 1. 本地编译

本地编译需要安装 vscode 及 相应插件，网上有教程。

搭建好环境后，用 vscode 打开本地的「PDF编译用源码/」内的 `main.typ` ，完成。

对了，还需要安装 Noto Serif CJK SC、Noto Sans CJK SC 这两款字体，或者修改代码，用本地其他字体代替。

### 2. 在线编译

需要注册 Typst 官方免费账号，无任何安装，手机上也能操作。

1. 注册免费账户：https://typst.app/ 
2. 新建项目：找到 Empty document 大按钮，点击，先随便输入个项目名称。
3. 上传资源：将本地「PDF编译用源码/」里的所有内容，上传至新建的项目。
4. 指定预览文件：在侧边栏选中 `main.typ` 文件，右键设为预览文件（或者点击“👁”图标）。
5. 完成：导出 pdf。

上传后，项目页面 Files 栏内，大概这样子：

```less
数据.txt
main_纯原文.typ
main_统计.typ
main_小段.typ
main.typ 👁
```

点亮各 `main_*.typ` 文件的👁，查看不同形式的输出。

这堆`main_*.typ`都是独立的。删除哪个都不影响其他的使用。

Typst 线上字体数量有限，把本地字体文件（如`*.ttf`、`*.otf`等）拖到 Files 栏内（在 Files 内就行，没有位置要求）， 代码`text(font:"...")`的弹出列表中就会出现相应字体名称，选择后，字体就会被应用。
